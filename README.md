# knnk-eda-reporting
Streamlit-based EDA reporting app with dynamic campaign mapping and automated insights for impressions, clicks, discrepancy analysis and CTR analysis.
##  Features
- Data Upload
-  Upload CSV / Excel files
-  Automatic data detection and preprocessing
-  Pivot Table Builder
-  Select dimensions & metrics dynamically
-  Auto aggregation (SUM)
-  Add TOTAL row automatically
-  Dynamic Campaign Mapping  
-  Automated EDA Reports
-  Date search option
-  KPI Analysis (Impressions, Clicks, CTR)  
-  Data Filtering & Visualization  
-  Fast & Interactive Dashboard using Streamlit
-  Get AI Summary Insights
-  Dynamic CPM Calculator
-  Google Sheets Integration 
- Store mappings in Google Sheets
- Persistent storage (no data loss)
- Works across sessions & users

---

##  Tech Stack

- **Frontend/UI**: Streamlit  
- **Data Processing**: Pandas, NumPy  
- **Visualization**: Streamlit Charts  
- **Storage**: Google Sheets API (gspread)  
- **Authentication**: Google Service Account
